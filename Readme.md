## State Machine/Button-Based Chatbot, made with Rasa
A simple variation of Rasa's MoodBot that uses buttons only.
To start:
- Train Model: rasa train
- Run Rasa Shell: rasa shell
- Run Action Server: rasa run actions
- Chat with MoodBot
